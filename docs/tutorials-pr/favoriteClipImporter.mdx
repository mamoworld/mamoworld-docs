# Favorite Clip Importer

Do you have some logos, intros or other clips you need in almost every Premiere Pro project? Learn how to write your own custom tool to do all of this in a single click: 
- import your favorite file in Pr
- place it in the active sequence
- modify clip properties like its position or scale

You even learn how to create a user interface for your custom importer tool, such that you can easily pick any of your favorite files:

<RetinaImg src="/img/tutorials-pr/favoriteClipImporter/favoriteClipImporterUI.png" alt="user interface of the favorite importer script"/>

After watching this tutorial, importing all your favorite footage items is just one click away!
