# Export Tools

## Export Footage List to Textfile {#export_list_of_footage_items_to_textfile}

### Usage

The tool **Export List of Footage Items to Textfile** exports the full media path of all footage items in your Premiere Pro project to a text file. You can also limit the export to only include
- files in a particular project bin
- selected files
- used files

<RetinaImg src="/img/tools-pr/exportTools/ExportListOfFootageToTextfile.png" alt="usage of the tool to export all Premiere Pro footage file paths to a text file"/>
